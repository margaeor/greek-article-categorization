Τρίτη 19 Σεπτεμβρίου 2000



ΣΤΗΛΕΣ









ΟΔΗΓΟΙ








Μια καμπάνια διαφορετική από τις άλλες ξεκινά εντός των ημερών, προκειμένου να γνωρίσουν οι Αθηναίοι το νέο αεροδρόμιο Σπάτων και να παίξουν το ρόλο του...κομπάρσου στις δοκιμές που θα γίνουν για τη λειτουργία του.

Με κίνητρα μια δωροεπιταγή αξίας 90.000 δραχμών και την περιέργεια απέναντι σε ένα σημαντικό δημόσιο έργο, η εταιρεία του αεροδρομίου Σπάτων θα προσκαλεί από τις αρχές Οκτωβρίου κάθε ενδιαφερόμενο να επισκεφθεί τα Σπάτα. Η ανοικτή πρόσκληση είναι αποτέλεσμα της ανάγκης να πραγματοποιηθούν εκτεταμένες δοκιμές για ένα εξάμηνο πριν ξεκινήσει η κανονική λειτουργία του αεροδρομίου (έχει προγραμματισθεί για την 1η Μαρτίου 2001).

Σύμφωνα με πληροφορίες, το κοινό θα προσκαλείται, με καταχωρίσεις στον Τύπο και γιγαντοαφίσες, να επισκεφθεί τις εγκαταστάσεις ενώ θα διατίθενται πούλμαν για τη μεταφορά του. Ανάμεσα στους πολίτες που θα φθάνουν στο νέο αεροδρόμιο θα γίνεται μία επιλογή με διάφορα κριτήρια (ηλικία, φύλο, επάγγελμα, τόπος κατοικίας, εισόδημα κ.λπ.) με στόχο να διαμορφωθούν ομάδες «αντιπροσωπευτικές» εκείνων που θα χρησιμοποιούν το αεροδρόμιο στην κανονική λειτουργία του. Στις παραπάνω πειραματικές ομάδες θα ενταχθούν 4.000 άτομα, τα οποία θα παραλάβουν και τις δωροεπιταγές των 90.000 δραχμών.

Οι επιλεγέντες θα καλούνται να βρίσκονται στα Σπάτα συγκεκριμένες ημέρες και ώρες, κατά τις οποίες θα γίνονται οι γενικές δοκιμές, για να διαπιστωθεί εάν οι χώροι είναι εξυπηρετικοί, εάν υπάρχουν κενά στις διαδικασίες προώθησης των επιβατών, εάν κάπου μπορεί να δημιουργηθούν εμπλοκές κ.ο.κ. Για τον ίδιο σκοπό, η εταιρεία του αεροδρομίου έχει αγοράσει 4.000 βαλίτσες τις οποίες θα δίνει στους πολίτες για να διαπιστώνεται εάν μπορούν να κινούνται απρόσκοπτα στους διάφορους χώρους.

Οι δοκιμές λειτουργίας του αεροδρομίου θα επεκταθούν αργότερα και σε άλλους χώρους. Οι αρχές ασφαλείας θα κληθούν να αντιμετωπίσουν πυρκαγιές, αεροπειρατίες, τρομοκρατικές επιθέσεις και την αναγκαστική προσγείωση αεροσκαφών. Σύντομα θα αρχίσουν επίσης και οι δοκιμαστικές πτήσεις κανονικών αεροσκαφών σε διάφορα μεγέθη. Με την ευκαιρία της ολοκλήρωσης των κατασκευαστικών έργων, η εταιρεία του αεροδρομίου πραγματοποιεί εκδηλώσεις στις 7 Οκτωβρίου, που θα περιλαμβάνουν φωτογραφική έκθεση για την εξέλιξη του έργου, επισκέψεις σε διάφορους χώρους, ομιλίες και προβολές βίντεο.

A.Χ.
